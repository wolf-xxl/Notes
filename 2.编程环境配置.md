#### 1. `Python`安装

1. [Python 官网| Python.org](https://www.python.org/downloads/)
2. [Anaconda](https://www.anaconda.com/download)
```bash
conda create --name [your-env-name] # 创建python环境
conda env list # 列出环境清单
conda env remove --name [your-env-name] # 删除python环境
conda activate [your-env-name] # 激活python环境
conda deactivate [your-env-name] # 失活python环境
```

#### 2. `JS`安装

[Node Version Manager 官网](https://nvm.uihtm.com/doc/download-nvm.html)
```bash
nvm install 18 # 安装 node.js 18
nvm list # 列出 node.js 版本清单
nvm use 18 # 激活 node.js 18
```
